## 変更ファイル

### `app/globals.css`
- Tailwind の base/components/utilities は維持
- Web1.0 用のカスタムクラスを追加:
  - `.retro-wrapper` — 白背景、ホットピンク右枠線、中央寄せ
  - `.retro-marquee` — 紺背景、黄色テキスト、スクロールアニメーション
  - `.retro-hr` — 紺の破線
  - `.retro-box` — 破線枠のコンテンツボックス
  - `.retro-btn` — 旧式ボタン風スタイル
  - `.retro-counter` — カウンター風数字表示

### `app/layout.tsx`
- `body` の背景色を `#E0FFFF` に変更
- フォントを MS Pゴシック に変更

### `app/page.tsx`（ログイン画面）
- 背景: `#E0FFFF`
- ラッパー: 白背景、右ピンク枠線
- ヘッダー: 「★☆ 振り返りアプリへようこそ！☆★」+ 破線区切り
- マーキー: 「今日も一日お疲れ様でした～♪ AIと振り返りをはじめよう！」
- GitHub・Google カード: 破線枠のボックスに変更
- ボタン: レトロなボタン風スタイル
- ゲストリンク: テキストリンク風

### `app/reflect/page.tsx`（振り返り画面）
- ヘッダー: 破線で区切る
- チャットバブル: ユーザーは `>` 引用符スタイル、AI は通常テキスト
- 入力欄: クラシックな input フィールド
- まとめ表示: `pre` タグ + 破線枠

### `app/history/page.tsx`（履歴画面）
- サイドバー: nav 風、破線右ボーダー
- 日付リスト: テキストリンク風ボタン
- コンテンツ: 白背景の `pre` タグ表示
